## Correr el código

Usa cmake para hacer una build del proyecto así en el folder raíz.

Crea una carpeta build
```
mkdir build
```

Te colocas en la carpeta
```
cd build
```

Ejecutas cmake
```
cmake ..
```


Esto creará un solución dependiendo de lo que se use, ya sea
- Visual Studio Code
- Visual Studio 2022
-
